# Magda.-Grafomotricidad-y-Pictogramas-Tic.-TEA.-
Grafomotricidad y Pictogramas TIC es un proyecto educativo dirigido principalmente a alumnado de FORMACIÓN BÁSICA OBLIGATORIA con nivel curricular de 3 años, orientado al desarrollo de la motricidad fina, la iniciación a la escritura y la comunicación visual mediante el uso de herramientas digitales e interactivas.
